Relációs Adatbázisok (SQL)

-- Komplex lekérdezés példa
SELECT u.name, COUNT(o.id) as total_orders
FROM users u
LEFT JOIN orders o ON u.id = o.user_id
WHERE u.created_at >= '2024-01-01'
GROUP BY u.id, u.name
HAVING COUNT(o.id) > 5;
  • ACID tulajdonságok
  • Komplex lekérdezések
  • Konzisztencia garantált
  • Érett eszközkészlet

NoSQL Adatbázisok

// MongoDB aggregation pipeline
db.users.aggregate([
  {$match: {created_at: {$gte: "2024-01-01"}}},
  {$lookup: {
    from: "orders", localField: "_id",
    foreignField: "user_id", as: "orders"
  }},
  {$project: {name: 1, order_count: {$size: "$orders"}}}
  • Horizontális skálázhatóság
  • Rugalmas séma
  • Nagy teljesítmény
  • Modern alkalmazásokhoz
